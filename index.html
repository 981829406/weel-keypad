最基本的用法

不需要任何额外的参数就可以将一个初始状态的“键盘”载入到页面中，点击动作栏的按钮操作显示/隐藏。 

// 使用 new 操作符得到一个实例
var kypd = new Keypad()

kypd.toggle() // 切换
kypd.show() // 显示
kypd.hide() // 隐藏
切换 显示 隐藏

绑定 & 监听 input 元素

使用 Keypad 不会阻止移动端浏览器对于 input 元素的默认事件而导致行为或者 显示异常（放大输入框、元素漂移），需要避免这种的话建议设置 'readonly'，具体参考下一示例。 

// 使用 new 操作符得到一个实例
var kypd = new Keypad({
  // 需要监听的 input 元素选择器
  el: '#listened-input'
})
切换 显示 隐藏
